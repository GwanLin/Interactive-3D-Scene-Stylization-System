# Interactive-3D-Scene-Stylization-System
Built an end-to-end 3D scene processing pipeline integrating Gaussian Splatting and neural style transfer  
•	Designed image processing pipeline (DoG, histogram equalization, morphology) for structural feature extraction  
•	Implemented geometry-aware masking techniques to reduce texture artifacts in 3D rendering  
•	Developed a real-time interactive application in Unity enabling dynamic scene transformation  
•	Tools: Python (OpenCV), Unity, MeshLab
